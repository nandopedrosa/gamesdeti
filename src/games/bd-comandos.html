Bancos de Dados - Comandos

Em sistemas gerenciadores de bancos de dados, é possível executar uma série de comandos com objetivos diferentes. Os comandos podem ser:

  • DDL - Data Definition Language
  • DML - Data Manipulation Language
  • DTL - Data Transaction Language
  • DCL - Data Control Language

Associe cada comando de banco de dados ao seu respectivo tipo.
