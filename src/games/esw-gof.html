Padrões de Projeto - Gang of Four

Um padrão de projeto descreve uma solução para resolver um problema genérico de projeto em um contexto específico. Ele dá nome, abstrai e identifica os aspectos-chave de uma estrutura de projeto comum para torná-la reutilizável.

Padrões de projeto podem ser Criacionais, Comportamentais ou Estruturais.

Associe cada padrão de projeto ao seu tipo e descrição.
